# LLD Projects

A collection of Low Level Design (LLD) projects implemented in Java.

## Overview

This repository contains various system design implementations and coding exercises focused on low-level design patterns and object-oriented programming concepts.

## Prerequisites

- Java 21 or higher
- Maven 3.6+

## Getting Started

### Clone the repository
```bash
git clone <repository-url>
cd LLD-Projects
```

### Build the project
```bash
mvn clean compile
```

### Run the application
```bash
mvn exec:java -Dexec.mainClass="org.example.Main"
```

## Project Structure

```
src/

│           └── example/
```
